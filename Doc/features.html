Features available

The SofaUnity plugin allows to use Sofa simulation engine inside Unity3D from several approaches.
  • Either by replacing the basic geometry shapes from Unity. See the section: Object Creation
  • By loading existing simulation scene (*.scn) from Sofa with several writting rules. See the section: SOFA scene parsing
  • Or by mixing the methods and using a simulation scene plus some other objects created in Unity. See the section: Mix methods

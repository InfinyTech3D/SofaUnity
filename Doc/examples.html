Demo Scenes

Caduceus
Caduceus opening scene from SOFA inside Unity using relfexion probes. Right is scene at start and left after simulation.

CollisionMultiple
Multiple collision of 3 deformable objects inside a wooden bowl.

Demo_fall_01
Basic example of two deformable bodies falling one upon the other and then on the floor.

Demo_fall_02
Same scene as demo_fall_01 but with an inclined plane before the floor.

SoftTissue
Simulation of soft tissue using triangular finite elements.

Tissue_fall
Combination of the SoftTissue scene and the SphereCollisionModel component applied on a Unity Cube 3DObject. Collision spheres are mapped onto the cube in order to collide with the tissue.

Liver
Classic simulation of Sofa flying Liver model using tetrahedron finite elements.

TriangleDetection
Ray casting demo scene showing several interactions possible between tools and a volumic model.

LiverInteraction
Combination of the Liver and triangleDetection scenes, allowing interactions between tools and the 3D liver model.


Benchmarks


A series of benchmarks scenes to show how to create objects and change their physical behaviour.


Surgery


Liver interacting with 3D physical tools. Simulated with SOFA framework inside Unity3D.
Youtube demo movie: Sofa in Unity - 3D Liver tickling


VROffice


Caduceus opening scene from SOFA in Virtual Reality.

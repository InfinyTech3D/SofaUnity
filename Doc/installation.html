Installation

  1. Install Unity3D version: 5.5.1 (version used for development) or higher.
  2. Create a new 3D project.
  3. In Unity3D Editor, use Assets -> Import Package -> Custom Package and load the package SofaUnityPlugin.unitypackage

  4. Know Bug: After importing the asset, you might have this error: DllNotFoundException: SofaAdvancePhysicsAPI SofaUnityAPI.SofaContextAPI..ctor () (at Assets/SofaUnity/Plugins/SofaUnityAPI/SofaContextAPI.cs:23)

    5. To resolve it, search for the file SofaAdvancePhysicsAPI.dll in the Unity project browser, then in its inspector click on reset. Finally, in the main menu of Unity click Assets->Reimport All.

  5. Know Bugs for this version: An absolute path is needed in one of the Demo scene. Edit the file: Assets/Scenes/SofaScenes/caduceus.scn and change line 18 to have fileTopology="{YourProjectDir}/Assets/Scenes/SofaScenes/mesh/snake_body.obj"

Compatibility

  • Requires Unity3D version: 5.5.1 or higher.
  • Provides, as Native plugin, SOFA version: 17.06
  • Environment: Currently supported only on Windows7 64bits.

VR asset

To use the scene that combine Sofa and Virtual Reality, several assets are needed:

  • OculusPlatform
  • OSPNative
  • OVR
  • OVRAudioManager
  • OvrAvatar
  • SteamVR
  • VRTK

Then, in the script SVRLaserRay.cs uncomment the line: "//#define USING_VRTK"
